# 更新履歷

## [1.7.11] - 2025-12-25

### ✨ 新增功能
- **節日祝福系統**
  - 好感度達到「朋友」以上的好友，會在節日當天發送祝福訊息到聊天室
  - 支援 8 個節日：元旦、農曆新年、西洋情人節、白色情人節、端午節、七夕、中秋節、聖誕節
  - AI 會根據角色個性和關係生成個性化祝福內容
  - 情人節類型節日對戀人關係有特別祝福

- **動態牆節日氛圍**
  - 節日當天，每個角色的第一則貼文會帶有節日氛圍
  - 之後的貼文恢復正常，不會洗版

- **Header 節日特效** 🎄
  - 情人節（2/14、3/14、七夕）：飄落愛心 ❤️💕💗
  - 農曆新年（除夕～初五）：飄落福字、紅包、金幣 福🧧💰🪙
  - 冬季（12/25～農曆新年前）：飄落雪花 ❄️❅❆

## [1.6.3] - 2025-12-08

- **修正記憶系統空回應未正確處理的問題**
  - 當 Gemini API 因安全機制封鎖內容時，記憶系統會收到空字串回應

  - 關係評估遇到空回應時會印出警告 log，並返回空陣列
  - 不影響主流程，但提供更清楚的錯誤診斷資訊

### 🔧 技術改進
- 記憶服務 API 呼叫後會檢查 `promptFeedback.blockReason` 和 `finishReason`
- 區分「真正的空回應」和「被安全機制封鎖」的情況
